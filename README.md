# Todo CLI
A simple command line todo application built with Python.
## Features
-Add a task
-Done a task
-List all tasks
-Delete a task
-Save all tasks to JSON file
## Installation
Install the project in editable mode:

```bash
pip install -e .
```
## Usage
Add a task:
```bash
todo add "learn readme"
```
List all tasks:
```bash
todo list
```
Delete a tast:
```bash
todo delete 1
```
Mark a task as done:
```bash
todo done 1
```
## Development
Install development dependnecies:
```bash
pip install -e ".[dev]"
```
Run tests:
```bash
pytest
```

## Data Storage
Tasks are stored in a local JSON file:
```text
www.json
```
This file is automatically when tasks are saved.
Example data:
```json
[
    {
        "a":1,
        "b":"wuyihan",
        "c":false,

    },
    {
        "a":2,
        "b":"wuyiqian",
        "c":false,
    }
]
```
Do not edit this file manually unless you are knowing what you are doing.
